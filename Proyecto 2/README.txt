Proyecto 2: Segmentación
Se requiere desarrollar e implementar un algoritmo de detección y conteo de leucocitos para el
diagnóstico de malaria mediante la técnica de gota gruesa (imágenes de gota gruesa). El algoritmo debe
generalizar el proceso de detección y conteo para todas las imágenes, esto quiere decir que, los
parámetros de cada etapa deben ser los mismos para todas las imágenes.
Se evaluará la correcta selección de las técnicas de procesamiento y segmentación de imágenes. El
algoritmo debe tener las siguientes 5 etapas para realizar la detección y conteo de leucocitos
generalizado: selección del espacio de color, filtrado, modificación de contraste , segmentación, y
morfología. Adicionalmente, se debe evaluar la etapa de detección del algoritmo (IoU).
Entregable
- Realizar una presentación con sustentación oral de máximo 15 minutos explicando el paso a paso
(diagrama de flujo o de bloques) y análisis del resultado de cada etapa implementada en la
imagen.
- Realizar un reporte escrito explicando el paso a paso (diagrama de flujo o de bloques) y análisis
del resultado de cada etapa implementada en la imagen.
- Adjunte en un comprimido la presentación, el reporte escrito y el código usado para las
modificaciones.
